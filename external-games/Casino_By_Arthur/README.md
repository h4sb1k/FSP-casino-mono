# Casino FSP MVP

MVP веб-приложения по ТЗ (быстрые игровые комнаты на бонусные баллы) с реализованной игровой оболочкой **«Взлом банка»**.

## Что реализовано

- Список комнат, фильтры и автоподбор комнаты.
- Ручной выбор комнаты.
- Резервирование бонусных баллов при входе.
- Кнопка буста (покупается один раз на раунд).
- Таймер ожидания перед стартом.
- Автозаполнение свободных мест ботами.
- Backend-логика выбора победителя (веса + RNG).
- Визуализация раунда «Взлом банка»:
  - все участники одновременно подходят к сейфу за 3.4-4.9 сек;
  - разные и непостоянные скорости взлома;
  - "ломаные туннели" к сейфу.
- История раундов и журнал backend-розыгрыша.
- Админ-конфигуратор комнаты с валидацией рисков.
- Страницы остальных игр с заглушкой: "Пока эта функция не готова".

## Структура (шаблон Vite + React)

- `index.html` — точка входа шаблона.
- `src/main.jsx` — bootstrap React.
- `src/App.jsx` — рендер legacy-разметки и подключение legacy-логики.
- `src/legacyMarkup.js` — HTML-структура MVP.
- `styles.css` — legacy-стили интерфейса (без изменения логики).
- `app.js` — legacy-логика rooms/matchmaking/round/config/history (без изменения логики).

2. `npm install`
3. `npm run dev`
4. Открыть `http://localhost:5173`

## Демонстрационный сценарий

1. Выберите фильтры и нажмите `Автоподбор комнаты`.
2. Нажмите `Войти в комнату (резерв)`.
3. По желанию нажмите `Купить буст`.
4. Нажмите `Запустить таймер ожидания`.
5. Дождитесь старта раунда, наблюдайте визуализацию и победителя.
6. Проверьте `Журнал backend-розыгрыша` и `Историю раундов`.
7. В `Конфигуратор комнаты (админ)` измените параметры и сохраните.
